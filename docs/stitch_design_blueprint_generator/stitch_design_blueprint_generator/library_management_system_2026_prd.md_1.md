# Library Management System 2026 PRD

## Executive Summary
Enterprise Library Management System for a single college.

### Roles
- Super Admin
- Admin
- Librarian
- Department Coordinator
- Student

### Key Features
- Physical & Digital Books
- QR/Barcode Tracking
- College Email Login
- Approval Workflow
- Fine Management
- Online/Offline Payments
- PDF Receipts
- Executive Dashboard
- Enterprise Reports
- Audit Logs
- Notifications (Email, SMS, WhatsApp, Push, In-App)
- Vendor & Procurement Management
- No Due Certificate
- PostgreSQL
- React + React Native + NestJS

### Confirmed Decisions
- Individual Copy Tracking
- Multi-language Support
- Enterprise Security + MFA
- Daily/Weekly/Monthly Backups
- Admin Configurable Renewals
- Digital Rights Management Per Book
